
# 89. “打字机”的QWERTY键盘布局

好的，宝贝，我们继续这个关于“意外发现”的故事。

你，有没有，仔细地，看过，我们家里的电脑键盘，或者，爸爸妈妈手机上的键盘呢？

你，有没有，发现，键盘上的字母，它们的排列顺序，非常非常奇怪？

最上面一排的字母，是Q、W、E、R、T、Y。

你，有没有，想过，为什么，键盘上的字母，不是，按照A、B、C、D、E、F的顺序，排列的呢？

它，也是，一个，非常非常有趣的“意外”！

这个故事的名字，就叫做《“打字机”的QWERTY键盘布局》。

在很久很久以前，大概，在1870年代的时候，世界上，第一台，可以，真正，用来打字的“打字机”，被发明出来了！

这台打字机，它，非常非常神奇！

它，有一个，一个的，小小的“金属杆”，叫做“字锤”。

当你，按下键盘上的一个字母的时候，那个字母，对应的“字锤”，就会，像一个小锤子一样，“啪”地一下，打到纸上，留下一个，清晰的字母！

但是，这台打字机，它，有一个，非常非常大的“麻烦”！

当人们，打字，打得，非常非常快的时候，尤其是，当他们，连续，打出，两个，经常，一起出现的字母的时候，比如，“th”或者“st”，那些，小小的“字锤”，它们，就会，互相，撞在一起，卡住！

打字机，就不能，继续，工作了！

这让发明打字机的工程师，他的名字，叫做克里斯托弗·莱瑟姆·肖尔斯，非常非常苦恼！

他，想：“我，发明打字机，是为了，让人们，打字，打得，更快！但是，现在，它，却因为，打字，打得太快，而卡住！这，可怎么办呢？”

他，想了，很多很多办法。

他，突然，有了一个，非常非常棒的主意！

他，想：“既然，打字，打得太快，会让字锤，卡住。那么，我，就让人们，打字，打得，慢一点，不就行了吗？”

于是，克里斯托弗·莱瑟姆·肖尔斯，他，就重新，排列了，键盘上的字母！

他，把那些，经常，一起出现的字母，比如，“th”或者“st”，它们，分得，远远的！

他，把那些，不经常，使用的字母，放在了，人们，最容易，按到的地方！

而把那些，经常，使用的字母，反而，放在了，人们，不太容易，按到的地方！

这样，人们，在打字的时候，就必须，把手指，移动得，更远，打字的速度，自然，就慢了下来！

那些，小小的“字锤”，它们，就不会，互相，撞在一起，卡住了！

这个，看起来，非常非常奇怪的键盘布局，就是我们现在，看到的，以最上面一排，前六个字母，命名的“QWERTY键盘”！

这个“QWERTY键盘”，它，虽然，不是，最快，也不是，最有效率的键盘布局。

但是，它，却成功地，解决了，当时，打字机，字锤，卡住的“大麻烦”！

所以，它，很快，就成了，全世界，打字机的“标准键盘”！

好了，宝贝，故事讲完了。

一个，本来，是为了，让人们，打字，打得，慢一点的“设计”，却意外地，成了，全世界，最最流行的键盘布局！

这个故事，告诉我们，有时候，那些，看起来，是“不完美”的设计，或者，那些，看起来，是“为了解决一个问题”的“妥协”，它们，里面，可能，都藏着，一个，非常非常大的“惊喜”！

它们，甚至，可以，影响我们，几百年，几千年，和科技，互动的方式！

现在，闭上眼睛，好好睡觉吧。也许在你的梦里，你也能像克里斯托弗·莱瑟姆·肖尔斯一样，用一个“小小的改变”，影响全世界。晚安。
