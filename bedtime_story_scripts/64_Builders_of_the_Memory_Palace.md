
# 64. “记忆宫殿”的建造者

好的，宝贝，我们继续这个关于记忆和宫殿的故事。

你有没有想过，我们的大脑，就像一个巨大无比的、装满了我们所有记忆的“仓库”？

但是，这个“仓库”，有时候，会有点乱。我们常常会找不到我们想要找的东西，比如，昨天老师讲的那个故事，或者，妈妈让我们去超市买的东西。

但是，在很久很久以前，有一些非常非常聪明的人，他们发明了一种神奇的方法，可以把我们大脑里的这个“仓库”，改造成一座宏伟的、整洁的、永远不会忘记任何东西的“记忆宫殿”！

这个故事的名字，就叫做《“记忆宫殿”的建造者》。

我们的故事，要从两千多年前的古希腊说起。

当时，有一位非常非常有名的诗人，他的名字，叫做西蒙尼德斯。

有一天，西蒙尼德斯，被邀请去参加一场非常盛大的宴会。宴会在一座宏伟的宫殿里举行。

在宴会上，西蒙尼德斯，为大家朗诵了一首他写的、非常优美的诗。朗诵完之后，他就先行离开了。

但是，就在他刚刚离开宫殿没多久，突然，“轰隆”一声巨响，那座宏伟的宫殿，竟然坍塌了！

所有还在宫殿里参加宴会的人，都不幸遇难了。

这是一场非常非常可怕的悲剧。更糟糕的是，所有遇难者的遗体，都已经被砸得面目全非，他们的家人，根本无法辨认出，哪一位，是自己的亲人。

就在所有人都束手无策的时候，有人想起了刚刚离开的诗人，西蒙尼德斯。他们把他，请回了废墟现场。

西蒙尼德斯看着眼前这片悲惨的废墟，他闭上了眼睛。

奇迹，发生了！

在他的脑海里，那座已经坍塌的宫殿，竟然完好无损地，重新浮现了出来！他清清楚楚地“看”到，在宴会大厅里，每一张桌子的位置；他也清清楚楚地“看”到，在每一张桌子后面，都坐着哪一位宾客！

他就这样，在自己的脑海里，重新“走”了一遍那座宫殿。然后，他睁开眼睛，准确地，说出了每一位遇难者的名字，和他们当时所在的位置！

所有的人，都为他那惊人的记忆力，感到了无比的震惊！

而西蒙尼德斯自己，也从这次悲剧的经历中，发现了一个关于我们大脑的、伟大的秘密：

我们的大脑，对于“地点”和“空间”的记忆，是天生就非常非常厉害的！

他想：“既然我们能如此轻松地，记住一个地方的样子。那我们能不能，把那些我们想要记住的、枯燥的东西，也‘放’到这个我们熟悉的地方里，然后，通过‘逛’这个地方，来把它们记住呢？”

这个伟大的想法，就是“记忆宫殿”的起源！

那么，我们普通人，要如何，在自己的大脑里，建造一座属于我们自己的“记忆宫殿”呢？

其实，非常非常简单！

**第一步：选择你的“宫殿”。**

你的“宫殿”，不需要真的像皇宫一样宏伟。它可以，就是你自己的家！因为，你的家，是你在这个世界上，最最熟悉的地方。你闭着眼睛，都能想起来，你的房间，是什么样子的。

**第二步：规划你的“参观路线”。**

你要在你的“宫殿”里，规划出一条固定的、永远不会改变的“参观路线”。比如，从你家的大门开始，然后是客厅，然后是厨房，然后是你的卧室，最后，是阳台。

**第三步：把你要记的东西，变成又夸张、又好笑的“图画”。**

我们的大脑，不喜欢记那些枯燥的文字和数字，但是，它非常非常喜欢记那些有趣的、夸张的、甚至有点傻乎乎的图画。

比如，今天，妈妈让你去超市，买三样东西：牛奶、面包和苹果。

你就可以，把它们，想象成三个非常夸张的画面：

想象一下，一大条“牛奶的河流”，正从你家的大门里，汹涌地流出来！

想象一下，你家客厅里的那个大沙发，竟然变成了一个巨大无比的、软绵绵的“大面包”！

想象一下，在你家厨房的桌子上，有好几个红色的“苹果”，正在那里，一边唱歌，一边玩杂耍！

**第四步：把这些“图画”，放到你的“参观路线”上。**

现在，你就可以，把你想象出来的这些好笑的画面，一个一个地，“放”到你之前规划好的那条“参观路线”上。

第一个地点，是你家的大门。你把“牛奶的河流”的画面，放在这里。

第二个地点，是你家的客厅。你把“大面包沙发”的画面，放在这里。

第三个地点，是你家的厨房。你把“会唱歌的苹果”的画面，放在这里。

**第五步：开始“参观”你的“记忆宫殿”！**

现在，你只要在你的脑海里，按照你规划好的路线，“走”一遍你的家。

当你“走”到大门口时，你就会“看”到那条牛奶的河流，你就会想起：哦，我要买牛奶！

当你“走”到客厅时，你就会“看”到那个大面包沙发，你就会想起：哦，我还要买面包！

当你“走”到厨房时，你就会“看”到那些会唱歌的苹果，你就会想起：啊，对了，还有苹果！

你看，就这么简单！你就把一个枯燥的购物清单，变成了一场有趣的、永远不会忘记的“宫殿之旅”！

好了，宝贝，故事讲完了。

“记忆宫殿”，是人类发明的一种非常非常古老，又非常非常厉害的记忆方法。

直到今天，全世界所有那些，能在几分钟之内，记住上千个随机数字，或者好几副扑克牌顺序的“记忆力冠军”，他们使用的，都是这个“记忆宫殿”的秘密武器！

这个故事告诉我们，我们每一个人的大脑里，都藏着一座等待我们去发现、去建造的“记忆宫殿”。

记忆，不是一件枯燥、痛苦的事情。它是一场，需要我们运用想象力、创造力和幽默感，去进行的、充满乐趣的游戏。

现在，闭上眼睛，好好睡觉吧。也许在你的梦里，你也能为自己，建造一座独一无二的、宏伟壮丽的“记忆宫殿”呢。晚安。
