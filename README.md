### **历史上的神秘与巧合 (Mysteries & Coincidences in History)**

21. **澳大利亚的“鸸鹋战争”**：澳大利亚军队如何带着机枪，向一群鸸鹋“宣战”，并最终“战败”。
22. **停不下来的疯狂舞会**：1518年，斯特拉斯堡的几百位市民为何突然开始疯狂跳舞，连续数周直到力竭而亡。
23. **两次卖掉埃菲尔铁塔的骗子**：一个名叫维克多·卢斯蒂格的男人，如何成功地将巴黎地标性建筑卖给废铁回收商。
24. **二战中的“幽灵军队”**：一支由艺术家和设计师组成的美国部队，如何用充气坦克和假声音骗过了德军。
25. **西伯利亚的神秘大爆炸**：1908年，通古斯地区发生了一场威力堪比千颗原子弹的爆炸，但至今没找到陨石坑。
26. **无人能懂的“伏尼契手稿”**：一本写满了奇怪文字、画满了奇特植物的古书，全世界最顶尖的密码学家都无法破译。
27. **“玛丽·赛勒斯特”号的幽灵船**：一艘完好无损的船漂在海上，船员却全部消失，食物和货物原封不动。
28. **林肯与肯尼迪的惊人巧合**：两位美国总统之间，存在着超过15个令人难以置信的巧合。
29. **消失的罗马第九军团**：一支五千人的精锐罗马军团，在进入苏格兰后，为何从历史上彻底蒸发了。
30. **铁面人**：路易十四时期，一个神秘囚犯为何终身佩戴天鹅绒（或铁质）面具，他的真实身份是什么？

### **改变世界的发明与工程 (World-Changing Inventions & Engineering)**

31. **伦敦“大恶臭”事件**：1858年伦敦的泰晤士河臭气熏天，如何催生了现代城市下水道系统的诞生。
32. **在沙漠中建造胡佛大坝**：在大萧条时期，成千上万的工人如何在极其艰苦的条件下，建成了这座“沙漠中的奇迹”。
33. **“协和”超音速客机**：工程师们如何让一架民航飞机飞得比子弹还快，从伦敦到纽约只需3个半小时。
34. **GPS的诞生**：最初为军事用途设计的全球定位系统，如何改变了我们每个人的生活。
35. **互联网的第一个“单词”**：1969年，第一台计算机试图向另一台发送“LOGIN”，但只成功发出了“LO”两个字母就崩溃了。
36. **巴拿马运河的“水楼梯”**：工程师们如何巧妙地利用船闸系统，让巨大的轮船“爬”过山脉。
37. **从海底连接世界**：工人们如何铺设横跨大西洋的海底电缆，第一次将欧洲和美洲用电报连接起来。
38. **乐高积木的“完美”精度**：一块小小的塑料积木，是如何做到拥有微米级的制造精度，让每一块都能完美拼接。
39. **“蓝色巨人”的国际象棋之战**：IBM的超级计算机“深蓝”，如何首次战胜了人类国际象棋世界冠军。
40. **维可牢尼龙搭扣的灵感**：一位瑞士工程师在遛狗后，如何从粘在狗毛上的苍耳中，发明了魔术贴。

41. **士兵“福泰克”熊**：一只叙利亚棕熊如何在二战中被波兰军队收养，并被正式授予士兵军衔，帮忙搬运炮弹。

49. **海豚“佩洛鲁斯·杰克”**：一头海豚在24年间，坚持为往返新西兰海峡的船只“领航”，从未出错。

55. **好莱坞标志的“前世今生”**：最初为了一个房地产广告而建的“HOLLYWOODLAND”标志，如何变成了世界电影之都的象征。

59. **莎士比亚的“失落岁月”**：这位伟大剧作家的生平中有七年时间完全没有记录，他究竟去了哪里？
60. **“图坦卡蒙的诅咒”**：打开这位年轻法老陵墓的探险队员和资助者，为何在之后几年内接连神秘死亡。

75. **“永不停歇”的雷暴**：委内瑞拉的卡塔通博河口，为何几乎每天晚上都会发生长达数小时的闪电。

### **生活中的意外发现 (Unexpected Discoveries in Daily Life)**

81. **“微波炉”的发明**：一位雷达工程师在工作时，发现口袋里的巧克力融化了，从而意外发明了微波炉。
82. **“便利贴”的诞生**：一位科学家试图发明一种超强力胶水，却失败地造出了一种粘不牢的弱胶水，结果成了办公神器。
83. **“X光”的发现**：伦琴在研究阴极射线时，无意中发现一种能穿透纸张和木板的神秘射线。
84. **“薯片”的由来**：一位厨师为了“报复”一位总嫌薯条切太厚的顾客，故意将土豆切成纸一样薄，炸脆后撒上盐，结果大受欢迎。
85. **“冰棒”的发明**：一个11岁的男孩在寒冷的夜晚，把一杯插着搅拌棒的苏打水忘在了门外，第二天早上得到了世界上第一根冰棒。
86. **“泡泡纸”的最初用途**：它最初被发明出来时，是想作为一种新潮的“立体壁纸”。
87. **“茶包”的意外走红**：一位茶叶商人为了方便，把茶叶样品装在小丝袋里寄给客户，客户却误以为要连袋子一起泡。
88. **“油炸圈饼”中间的洞**：一位船长为了在风浪中能把面包固定在舵轮上，用手杖在面团中间戳了一个洞。
89. **“打字机”的QWERTY键盘布局**：这种奇怪的字母排列，最初是为了防止打字员打得太快导致机械卡住而设计的。
90. **“安全玻璃”的灵感**：一位法国科学家失手打碎了一个玻璃烧瓶，发现瓶子虽然裂了但没有碎成一地，因为瓶内一层薄膜粘住了碎片。
